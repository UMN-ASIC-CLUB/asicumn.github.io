Beginner · Recommended First

RISC-V Processor Core

Build a working RISC-V CPU from scratch in Verilog. Start with individual logic gates, build up to a complete RV32I processor, simulate it in Vivado, and understand how real processors execute instructions. This is the most important project to start with — everything else builds on this foundation.

Tools: Vivado, ModelSim
Language: Verilog
Hardware: Basys3 FPGA (optional)
→ projects/riscv-core
Intermediate

NVDLA MAC Array

Reimplement the convolution MAC array from NVIDIA's open-source Deep Learning Accelerator, resource-constrained to fit an Artix-7 FPGA. Understand how neural network hardware actually works at the silicon level.

Prereq: RISC-V track
Ref: NVDLA GitHub
→ projects/nvdla-mac-array
Advanced

Chiplet NoC

Design a network-on-chip interconnect for chiplet communication using the UCIe standard. This is the frontier of semiconductor architecture — modular, reusable silicon blocks that talk to each other on a silicon interposer.

Prereq: Both tracks above
Standard: UCIe
→ projects/chiplet-noc
