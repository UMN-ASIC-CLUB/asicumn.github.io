01 / PROJECT

RISC-V Processor
Core.

Featured Active Open Source All Levels

5-Stage Pipelined
RV32I Core

Build a working RISC-V CPU from scratch in Verilog — IF, ID, EX, MEM, WB — with hazard detection, data forwarding, and branch handling. Simulate in Vivado, validate on FPGA, then trace the path from RTL through synthesis and place-and-route toward an actual tapeout. This is the foundation for everything else.

LanguageVerilog / SystemVerilog
ToolsVivado · Synopsys DC · Cadence Innovus
Process TargetSky130 (SkyWater 130nm)
HardwareBasys3 / Arty FPGA (optional)
ISARV32I → RV64I
PartnerChipFoundry / chipIgnite

rv32i_pipeline.sv 
// 5-stage RV32I — Pipeline Core
module rv32i_core #(
  parameter int XLEN = 32
) (
  input  logic        clk,
  input  logic        rst_n,
  output logic [XLEN-1:0] dbg_pc,
  output logic        dbg_valid
);

// ── IF/ID pipeline register ──
logic [XLEN-1:0] if_id_pc, if_id_instr;

always_ff @(posedge clk or negedge rst_n)
  if (!rst_n) {if_id_pc,if_id_instr} <= '0;
  else if (!stall) {
    if_id_pc    <= pc;
    if_id_instr <= imem_rdata;
  }

// ── Hazard + Forward Unit ──
hazard_unit u_haz (
  .rs1_id(id_rs1),
  .rs2_id(id_rs2),
  .rd_ex (ex_rd),
  .rd_mem(mem_rd),
  .stall (stall),
  .flush (flush)
);
endmodule
