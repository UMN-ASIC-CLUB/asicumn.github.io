# UMN ASIC Club

**University of Minnesota's home for semiconductor design — ASIC, FPGA, chiplet architecture, and beyond.**

We exist to close the gap between ECE coursework and real hardware engineering.

---

## Active Projects

| Project | Difficulty | Description |
|---|---|---|
| `riscv-core` | Beginner | RV32I processor core in Verilog, simulated in Vivado |

---

## Tools & Resources

- **HDL:** Verilog / SystemVerilog
- **Simulation & Synthesis:** Vivado (Xilinx), ModelSim
- **Professional EDA:** Cadence (available to members)
- **Fabrication:** Cleanroom training through UMN, Tiny Tapeout for actual silicon
- **Reference Architectures:** [PicoRV32](https://github.com/YosysHQ/picorv32), [CVA6](https://github.com/openhwgroup/cva6)
